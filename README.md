# JobFit

See how well your resume matches a job — before you spend an evening applying.

Paste your resume and one or more job postings. JobFit ranks your match, shows the
exact keywords worth adding, and can check your resume's overall health.
Works for any role, any field, any country.

- **No install.** Open it in any browser, including on a phone.
- **Private.** Everything runs on your device. Nothing is uploaded or saved.
- **Free.** No account, no servers, no API keys.

JobFit suggests words that are honestly yours to add — it never invents experience.

## Features
- Smart keyword matching with synonyms (JS = JavaScript) and UK/US spelling (organise = organize)
- Must-have vs. nice-to-have detection, with job-title and requirement weighting
- Rank several job postings at once, best-fit first
- Resume health check (contact info, quantified results, weak phrases, clichés, sections)
- "Try an example" samples across five fields
- Interface in English, Spanish, Hindi, French, and Arabic (right-to-left)
- Light and dark mode

## Use it
Open `index.html`, or visit the hosted link.

## Host it (free, on GitHub Pages)
1. Put `index.html` in a public GitHub repository.
2. Settings → Pages → Deploy from a branch → branch `main`, folder `/ (root)` → Save.
3. Your public link appears at `https://<username>.github.io/<repo>/`.

The file must be named `index.html` exactly.
